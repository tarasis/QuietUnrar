QuietUnrar by tarasis

A tool to quietly extract RAR files for Mac OS X.

A tool that quietly extracts RAR compressed files and only bothers the user if some interaction is required like a missing part or a password. The project was bourne out of being frustrated with UnRarX which displays a dialog of the progress and leaves it on the screen at the end of unarchiving the file.

Dependencies

The Unrar library provided by http://rarlabs.com. I have downloaded The most recent version 3.9.6 http://www.rarlab.com/rar/unrarsrc-3.9.6.tar.gz and it is included with the project with the permission of the readme.txt & license.txt
